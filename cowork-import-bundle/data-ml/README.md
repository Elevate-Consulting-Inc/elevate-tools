# Data & ML Pack

9 skills inspired by leading AI / data-science researchers.

## What's inside

| ZIP | Perspective |
|---|---|
| `dataanalysis-bengio.zip` | Deep learning foundations (Yoshua Bengio) |
| `dataanalysis-goodfellow.zip` | GANs and generative models (Ian Goodfellow) |
| `dataanalysis-hinton.zip` | Neural nets, backprop, the big ideas (Geoffrey Hinton) |
| `dataanalysis-lecun.zip` | Self-supervised learning, computer vision (Yann LeCun) |
| `dataanalysis-li.zip` | Computer vision and AI ethics (Fei-Fei Li) |
| `dataanalysis-ng.zip` | Practical ML and AI education (Andrew Ng) |
| `dataanalysis-oneil.zip` | Algorithmic fairness and harm (Cathy O'Neil) |
| `dataanalysis-rudin.zip` | Interpretable models over black boxes (Cynthia Rudin) |
| `dataanalysis-thrun.zip` | Self-driving, robotics, applied AI (Sebastian Thrun) |
