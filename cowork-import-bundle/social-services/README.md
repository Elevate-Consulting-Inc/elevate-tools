# Social Services Pack

9 skills with perspectives from across a Canadian social services agency.

## What's inside

| ZIP | Perspective |
|---|---|
| `social-boardceo.zip` | Agency CEO. Systems integrator across silos. |
| `social-boardchair.zip` | Board chair. Executive-level oversight. |
| `social-boardtreasurer.zip` | Board treasurer. Funding cycles and financial scrutiny. |
| `social-casemanager.zip` | Frontline case manager. Real-world policy reality. |
| `social-clinicallead.zip` | Clinical lead, mental health & addictions. Evidence-led care. |
| `social-firstresponder.zip` | Fire/EMS first responder. Crisis and overdose response. |
| `social-lawenforcement.zip` | Police sergeant. Community policing & mental health liaison. |
| `social-outreachworker.zip` | Community outreach. Trust-building, shelter navigation. |
| `social-policydirector.zip` | Provincial policy director. Inter-ministry coordination. |

## How to install

See **START-HERE.md** in the parent folder.

## Not sure which first?

- Working with clients directly? `social-casemanager.zip`
- Working on policy? `social-policydirector.zip`
- Governing an agency? `social-boardceo.zip`
