# Security & Reliability Pack

6 skills for keeping apps safe and running.

## What's inside

| ZIP | What it does |
|---|---|
| `tiger-appsec.zip` | App security engineer. Secure design, secrets, hardening. |
| `tiger-chaos.zip` | Site reliability engineer. SLOs, chaos testing, incident response. |
| `tiger-memory.zip` | Memory leak hunter. Finds apps that eat RAM. |
| `tiger-pentest.zip` | Red team lead. Thinks like an attacker. |
| `tiger-performance.zip` | Performance engineer. Finds slowness. Fixes tail latency. |
| `tiger-qa.zip` | QA lead. Test strategy, automation, release gates. |
