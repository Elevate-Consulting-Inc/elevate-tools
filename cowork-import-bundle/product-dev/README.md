# Product & Dev Pack

6 skills for people building software or products.

## What's inside

| ZIP | What it does |
|---|---|
| `devops-creativedirector.zip` | Reviews visuals for brand consistency and design quality. |
| `devops-developer.zip` | Senior full-stack developer. Ships real code, fast. |
| `devops-engineer.zip` | Platform and reliability engineer. Thinks in tradeoffs. |
| `devops-productowner.zip` | Product owner. Keeps the team focused on outcomes. |
| `devops-uxexpert.zip` | UX expert. Accessibility, information architecture, service design. |
| `devops-uxresearcher.zip` | UX researcher. Tests assumptions before you build. |

## How to install

See **START-HERE.md** in the parent folder.

## Not sure which first?

Start with `devops-productowner.zip`. It helps you plan before building.
