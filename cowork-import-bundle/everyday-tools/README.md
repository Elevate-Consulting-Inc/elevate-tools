# Everyday Tools Pack

Two skills that are useful for anyone. Start here if you're new.

## What's inside

| ZIP | What it does |
|---|---|
| `gail.zip` | Gail — a tough-love money coach. Ask about budgets, debt, or where your money went. |
| `humanize.zip` | Fixes writing that sounds like it came from a robot. Paste in text, get back natural-sounding writing. |

## How to install

See **START-HERE.md** in the parent folder.

## Not sure which first?

Try `humanize.zip`. You'll use it the most.
