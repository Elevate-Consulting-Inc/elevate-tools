# Medical Ethics Pack

6 skills with perspectives on healthcare ethics across BC and Alberta.

## What's inside

| ZIP | Perspective |
|---|---|
| `medicalethics-acutecare.zip` | Hospital ethicist. ICU, oncology, transplant decisions. |
| `medicalethics-continuingcare.zip` | Long-term care administrator. Dignity, restraints, resident rights. |
| `medicalethics-ems.zip` | EMS supervisor. Field consent, involuntary transport, scene safety. |
| `medicalethics-indigenous.zip` | Indigenous health partner. Cultural safety, data sovereignty. |
| `medicalethics-publichealth.zip` | Public health ethicist. Outbreaks, surveillance, policy. |
| `medicalethics-rural.zip` | Rural family doctor. Ethics under resource constraints. |

## How to install

See **START-HERE.md** in the parent folder.
