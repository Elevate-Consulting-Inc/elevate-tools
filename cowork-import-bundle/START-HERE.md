# Add Expert Skills to Your Claude

Hi! This folder has a bunch of skills you can add to Claude.

**What's a skill?** A small file that turns Claude into a specific kind of expert. Once you add one, Claude can think like that expert when you ask.

You do **not** have to install them all. Just grab the ones you'll use.

---

## Step 1 — Pick a pack

Each folder below is a "pack". Open the one that matches you.

| Folder | Who it's for | How many skills |

| `medical-ethics/` | Healthcare ethics and policy | 6 |

**Not sure?** Start with `everyday-tools/`. Small and handy.

---

## Step 2 — Upload one skill (practice round)

Do this **one time** to learn how. Then repeat for each skill you want.

1. Go to **https://claude.ai** and log in.
2. Click your **name** in the bottom-left corner.
3. Click **Settings**.
4. Click **Capabilities** on the left side.
5. Scroll down until you see **Skills**.
6. Click **Upload skill**.
7. A file picker opens. Find one `.zip` file inside your pack folder.
8. Select it. Click **Open**.
9. Wait a couple seconds. The skill shows up in your list.

**Done.** That's one skill added.

---

## Step 3 — Repeat for each ZIP you want

Same steps as above. Different ZIP each time.

Tip: do 3 or 4 at a time, then take a break. No need to marathon.

---

## Step 4 — Check it worked

1. Open a new chat: **https://claude.ai/new**
2. Type: `what skills do you have?`
3. Claude lists them. If you see the ones you added — you did it.

---

## Quick tips

- **One ZIP = one skill.** The filename tells you the skill's name.
- **Don't install stuff you won't use.** Just clutter.
- **You can add more later.** No rush.
- **Wrong one?** Go back to Skills. Click the trash icon next to it. Gone.

---

## If something breaks

| Problem | Try this |
|---|---|
| "Upload skill" button is missing | Your account may not have this feature yet. Ask Justin. |
| Skill doesn't show up after upload | Refresh the page. |
| "Invalid skill" error | Re-download that ZIP — it may have gotten corrupted. |
| The skill exists but Claude ignores it | Start a **new** chat. Skills only load in new conversations. |
